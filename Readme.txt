To run this program, just do:
	make

If library "graphics.h" is not installed and you don't wan't to use it,
	remove "#include <grapics.h>" into "main.c"
	remove " -lgraph" into "Makefile"
	remove all the "draw" function into "main.c"
	remove "draw();" into "main" function of "main.c"

To change the values of parameters, replace them into the "main" function into "main.c"
